# Longbridge Investors

A tri-lingual static site introducing famous investors, their holdings (SEC 13F or curated), and downloadable Claude Code skills.

## Quick Start

```bash
bun install
bun run fetch:data
bun run dev
```

Visit http://localhost:5173.

## Build

```bash
bun run build
```

Static output in `dist/`.
